Creates a new set of NativeContextExtensions for the Move VM, configuring extensions such as ObjectRuntime and NativesCostTable. These extensions manage object resolution, input objects, metering, protocol configuration, and metrics tracking. They are available and mainly used in native function implementations via NativeContext instance.
