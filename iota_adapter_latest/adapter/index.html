The VM is configured using a VMConfig that sets limits for vector length, value depth, and other runtime options based on the provided ProtocolConfig. These extensions manage object resolution, input objects, metering, protocol configuration, and metrics tracking. They are available and mainly used in native function implementations via NativeContext instance.
