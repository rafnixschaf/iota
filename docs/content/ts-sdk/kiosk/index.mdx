# Kiosk SDK

Kiosk SDK is a tool to interact with IOTA Kiosk. You can use it to query kiosk related data,
build transactions to interact, and extend Kiosk to support custom rules.

## Installation

```sh npm2yarn
npm i @iota/kiosk
```

## About

To use the Kiosk SDK, you must create a `KioskClient` instance. This instance is used to hold the
state of `IotaClient`, `Network`, and the rules and package IDs for interacting with IOTA Kiosk
extension.

-   [Read more about `KioskClient`](./kiosk-client/introduction.mdx)
