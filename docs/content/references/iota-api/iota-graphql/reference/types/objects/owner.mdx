An Owner is an entity that can own an object. Each Owner is identified by a IotaAddress which
represents either an Address (corresponding to a public key of an account) or an Object, but
never both (it is not known up-front whether a given Owner is an Address or an Object).


```graphql
type Owner implements IOwner {
  address: IotaAddress!
  objects(
    first: Int
    after: String
    last: Int
    before: String
    filter: ObjectFilter
  ): MoveObjectConnection!
  balance(
    type: String
  ): Balance
  balances(
    first: Int
    after: String
    last: Int
    before: String
  ): BalanceConnection!
  coins(
    first: Int
    after: String
    last: Int
    before: String
    type: String
  ): CoinConnection!
  stakedIotas(
    first: Int
    after: String
    last: Int
    before: String
  ): StakedIotaConnection!
  defaultIotansName(
    format: DomainFormat
  ): String
  iotansRegistrations(
    first: Int
    after: String
    last: Int
    before: String
  ): IotansRegistrationConnection!
  asAddress: Address
  asObject: Object
  dynamicField(
    name: DynamicFieldName!
  ): DynamicField
  dynamicObjectField(
    name: DynamicFieldName!
  ): DynamicField
  dynamicFields(
    first: Int
    after: String
    last: Int
    before: String
  ): DynamicFieldConnection!
}
```

### Interfaces

#### [`IOwner`](/references/iota-api/iota-graphql/reference/types/interfaces/iowner) <Badge class="badge badge--secondary" text="interface"/> 
> 
> 
> Interface implemented by GraphQL types representing entities that can own objects. Object owners
> are identified by an address which can represent either the public key of an account or another
> object. The same address can only refer to an account or an object, never both, but it is not
> possible to know which up-front.
>
