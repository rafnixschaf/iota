---
title: IOTA SDKs
---

IOTA provides developer kits that act as wrappers for the IOTA API.

## IOTA dApp Kit

[IOTA dApp Kit](ts-sdk/dapp-kit/index.mdx) is a web frontend SDK that interacts with the IOTA API. It is available as an NPM package.

## IOTA Rust SDK

See [Rust SDK](./rust-sdk.mdx) for SDK configuration and examples of using the IOTA API with Rust.

## IOTA TypeScript SDK

The IOTA TypeScript SDK documentation is available in its own microsite: [IOTA Typescript SDK](ts-sdk/typescript/index.mdx).


### Crate documentation


[GitHub Pages](https://iotaledger.github.io/iota/iota_sdk/index.html) hosts the generated documentation for Rust crates in the IOTA repository.
