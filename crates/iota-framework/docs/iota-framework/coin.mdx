## Function `treasury_into_supply`

Unwrap <code>
<Link to="../iota-framework/coin#0x2_coin_TreasuryCap">TreasuryCap</Link></code> getting the <code>
Supply</code>.

Operation is irreversible. Supply cannot be converted into a <code>
<Link to="../iota-framework/coin#0x2_coin_TreasuryCap">TreasuryCap</Link></code> due
to different security guarantees (TreasuryCap can be created only once for a type)


<pre><code>
<b>public</b> <b>fun</b> <Link to="../iota-framework/coin#0x2_coin_treasury_into_supply">treasury_into_supply</Link>&lt;T&gt;(treasury: <Link to="../iota-framework/coin#0x2_coin_TreasuryCap">coin::TreasuryCap</Link>&lt;T&gt;): <Link to="../iota-framework/balance#0x2_balance_Supply">balance::Supply</Link>&lt;T&gt;
</code></pre>



<details>
<summary>Implementation</summary>


<pre><code>
<b>public</b> <b>fun</b> <Link to="../iota-framework/coin#0x2_coin_treasury_into_supply">treasury_into_supply</Link>&lt;T&gt;(treasury: <Link to="../iota-framework/coin#0x2_coin_TreasuryCap">TreasuryCap</Link>&lt;T&gt;): Supply&lt;T&gt; \{
