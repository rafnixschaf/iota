## Resource `Clock`

Singleton shared object that exposes time to Move calls.  This
object is found at address 0x6, and can only be read (accessed
via an immutable reference) by entry functions.

Entry Functions that attempt to accept <code>
<Link to="../iota-framework/clock#0x2_clock_Clock">Clock</Link></code> by mutable
reference or value will fail to verify, and honest validators
will not sign or execute transactions that use <code>
<Link to="../iota-framework/clock#0x2_clock_Clock">Clock</Link></code> as an
input parameter, unless it is passed by immutable reference.
