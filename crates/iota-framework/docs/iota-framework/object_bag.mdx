<Link id="0x2_object_bag"/>
Similar to <code>
iota::bag</code>, an <code>
<Link to="object_bag#0x2_object_bag_ObjectBag">ObjectBag</Link></code> is a heterogeneous map-like collection. But unlike
<code>
iota::bag</code>, the values bound to these dynamic fields _must_ be objects themselves. This allows
for the objects to still exist in storage, which may be important for external tools.
The difference is otherwise not observable from within Move.
